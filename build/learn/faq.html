Is this going to replace Liferay Portal?

No, they complement each other. The Launchpad Project aims to be your project's back-end for modern applications, allowing you to create RESTful APIs instantly, powerful real-time cloud database, host static files, and leverage custom real-time capabilities. Regardless of what platform you work with, you'll be able to use native SDKs for mobile, desktop, and IoT devices.

How is this going to be integrated with Liferay Portal?

Launchpad can be described as Liferay's API Gateway that can also be used standalone. Launchpad provides an infrastructure to deploy micro-services that integrates with any other platform. There are several Liferay integrations in the works. Stay tuned.

Can I rely on Launchpad for apps in production?

The closed beta version of Launchpad is intended to collect developers feedback. We don't recommend you use it for production at this point.

Can Launchpad be installed on my own servers?

Not at this moment. Launchpad's initial goal is to provide platform and software as a service. Most of the complexity, such as auto-scaling your applications, is automatically provided by our infrastructure. In order to facilitate development, we have provided a server SDK, which you can run locally on your computer.
